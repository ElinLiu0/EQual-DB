GPU Accelerate Data Storage Based On Python

EqualDB is a Python Language and Rapids API Based Database.But Inspired by the power of GPU Acceleration and panel data,The EqualDB now is right here.Design for HPC and High Density Computing.

Intro

A better way to high density and data storage

"The more you buy,the more you save".This means to GPUs,but not means to EqualDB.EqualDB is fully opensource platform based on Python Rapids API and CUDA,this means EuqalDB is super easy for DataScientist and DataEngineer to get GPU Horse Power.

Speed

Storaged on disk,Transfer in GPUs.By using regularly CPU I/O,to using GPUs into increasing the Parallel Computbility and I/O BandWith.

Debug

Based On Python,Thats means EqualDB is super easy for beginner,data scientist and structure engineers.

Ecology

Based on NVIDIA CUDA Platform,which means you can use the feature like nv-link to Multi up the GPU Power or using BlueField™ DPU to make your platform more security.

Augmentability

Based Python CUDA,this means you can use C/C++ to write the plugins for EqualDB,otherwise you can use Cython or other Py2C Tools to compile into C based files.

Future RoadMap

What we will do in future?

  • Let the project run regularly and keep up with the latest updates.

  • Let EqualDB can run with JS,Python,C/C++,Java and more Languages.

  • Let EqualDB can run with GPU,DPU and more Devices.With NV-LINK™ and BlueField™ Stack.

  • Let EqualDB be cloud native GPU native database.
